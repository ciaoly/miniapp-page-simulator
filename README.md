# miniapp-page-simulator

一个web程序, 可以把微信小程序(或支付宝小程序也可以)的page.wxml代码渲染成网页. 配合PWA效果更佳.

**(代码来源于网络, 如果有任何不合适的言论或内容请提Issue)**


**本库只用来方便调试界面, 方便使用devtool调试css, 千万不要用来行违法犯罪之事!!!**
